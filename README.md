<h1>About this project</h1>
<br>
<p>사용자 입장에서 홈페이지를 조금더 사용하기 쉽게 UI/UX 디자인를 변경하여 대구백화점 페이지를 리뉴얼 하였습니다.</p>
<p>기존의 페이지가 버튼과 글자가 작아 가독성이 떨어지는 부분을 보완하여 백화점 특유의 시원하고 깔끔한 느낌을 내고자 하였습니다.</p>
